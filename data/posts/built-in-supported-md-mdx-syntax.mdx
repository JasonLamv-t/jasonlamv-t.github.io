### Basic inline syntax

This Blog was based on **Tailwind Next Blog**, a blog template open source project at [Github](https://github.com/jasonlamv-t/tailwind-next-blog). You can use this template via run `npx degit https://github.com/JasonLamv-t/tailwind-next-blog` in a empty directory to initialize for your own blog.

### Header

The above header was `h3`, your can define a header like this: 

```MDX
# This is H1
## This is H2
### This is H3
#### This is H4
##### This is H5
###### This is H6
```

And the effect like this:

# This is H1
## This is H2
### This is H3
#### This is H4
##### This is H5
###### This is H6

<Info>Header 2&3 has auto-gen link, your can hover to check the effect.</Info>

### Lists

#### Ordered

```md
1. lv_1 
2. lv_2
   1. lv_2_1
      1. lv_2_1_1
   2. lv_2_2
3. lv_3
```

1. lv_1 
2. lv_2
   1. lv_2_1
      1. lv_2_1_1
   2. lv_2_2
3. lv_3
  
#### Unordered

```md
- Item 1
- Item 2
  - Item 2_1 in nested
    - Item 2_1_1 in nested
  - Item 2_2 in nested
- Item 3
```

- Item 1
- Item 2
  - Item 2_1 in nested
    - Item 2_1_1 in nested
  - Item 2_2 in nested
- Item 3

#### Tasklist

```md

- [x] done
```

- [x] done

#### With title

````md
```ts:Title

const title = 'Code blocks with title';
```
````

```
````
